Nice to meet you!

My name is Wes, and I am a software developer with a love for game development.

I got sick of making boring command line programs and decided to level up my skills by learning about graphical libraries.

Over time, I've tried my hand at a bunch of different game engines using various languages and scripting techniques.

One of the things I'm most proud of is this 3D online multiplayer chess game I made all by myself.

I even designed all the assets using Blender. It was a ton of work, but I still remember the satisfaction and excitement I felt when I had it working.

I love the creative freedom that software development offers. Sure, it can be frustrating when something doesn't work correctly, but there's nothing like the feeling of making a small win that eventually adds up to something amazing.

Projects

In the beginning...

The first graphical game I made was pong, and the inspiration behind the page logo. Unfortunately, this was before I knew how to use github to host my projects, and I seem to have lost the files. This was still a great learning experience. I ended up remaking the game here in a few hours using p5 library.

I originally started with a perfect AI, but that's boring, so now it makes a mistake every once in a while. If you play long enough, you might notice some details like the ball is slowly speeding up. Or, how the AI gracefully returns to the center after each hit, and waits for you to strike back before moving!

Instructions: click on the game window, and use W and S keys to move up and down, and spacebar to start the ball. If you are on a mobile device, you can toggle the UI slider on for touch control. Check out the options panel below the game!
