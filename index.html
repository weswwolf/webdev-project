My name is Wes, a software developer with a love for game development. I got sick of making boring command line programs and decided to level up my skills by learning about graphical libraries. That's when I discovered the SDL library for C++ which allowed me to make simple games. Over time, I've tried my hand at a bunch of different game engines using all sorts of languages and scripting techniques. I've used Unity (C#), Godot (GDScript), Bevy (Rust), and GameMaker Language (GML) to name a few. One of the things I'm most proud of is this 3D online multiplayer chess game I made all by myself. I even designed all the assets using Blender. It was a ton of work, but I still remember the satisfaction and excitement I felt when I had it working. I love the creative freedom that software development offers. Sure, it can be frustrating when something doesn't work correctly, but there's nothing like the feeling of making a small win that eventually adds up to something amazing.
